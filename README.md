# بوت المكتبة الإلكترونية للأرشفة والتوثيق
# Electronic Library Bot for Archiving and Documentation

بوت تليجرام مصمم لتسهيل عمليات الأرشفة والتوثيق، يتضمن نظام رموز دخول مزدوج (مستخدم/مشرف)، تقارير إحصائية، ونظام قولبة ذكي.

## المميزات

- **نظام رموز الدخول المزدوج**: رمز للمستخدم العادي ورمز للمالك/المشرف
- **قاعدة بيانات SQLite**: لتخزين التوثيقات والمشرفين
- **شجرة أزرار تفاعلية**: تنقل عبر الأشهر الهجرية → الأسابيع → أنواع الأنشطة
- **نظام الأرشفة**: رفع النصوص والوسائط (صور/فيديو/PDF) وحفظها
- **لوحة تحكم المشرف**: سحب البيانات المؤرشفة حسب الشهر والأسبوع
- **إشعارات فورية**: إرسال إشعار لجميع المشرفين عند كل عملية أرشفة
- **التقارير الإحصائية**: تقارير شهرية مفصلة حسب الأسابيع والأنشطة
- **نظام القولبة الذكي**: دمج الصور مع شعار الجهة تلقائياً

## التثبيت والإعداد

### 1. استنساخ المشروع

```bash
git clone https://github.com/abdhaljorashy-arch/archive-bot.git
cd archive-bot
```

### 2. إنشاء بيئة افتراضية

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 4. الإعداد

افتح ملف `bot.py` وعدّل التالي:

- `API_TOKEN`: توكن البوت من BotFather
- `USER_PASSWORD`: رمز الدخول للمستخدمين العاديين
- `OWNER_PASSWORD`: رمز الدخول للمالك/المشرف

### 5. إضافة الشعار (اختياري)

ضع ملف `logo.png` في نفس مجلد البوت لتفعيل ميزة القولبة بالشعار.

### 6. تشغيل البوت

```bash
python bot.py
```

## طريقة الاستخدام

1. أرسل `/start` للبوت
2. أدخل رمز الدخول:
   - `user123` للمستخدم العادي (أرشفة فقط)
   - `owner123` للمالك/المشرف (أرشفة + سحب بيانات + تقارير)
3. اختر من القائمة الرئيسية:
   - **البدء في الأرشفة**: اختر الشهر → الأسبوع → نوع النشاط → أرسل المحتوى
   - **التقارير الإحصائية**: اختر الشهر لعرض إحصائيات مفصلة
   - **نظام القولبة**: أرسل صورة ليتم دمجها مع الشعار
   - **لوحة المشرفين**: سحب جميع البيانات المؤرشفة لفترة محددة

## ملاحظات تقنية

- **تخزين الملفات**: يحفظ البوت `file_id` من تليجرام بدلاً من تحميل الملفات لتوفير المساحة
- **الصلاحيات**: المشرفون يُحفظون في قاعدة البيانات وتبقى صلاحياتهم دائمة
- **المكتبة المستخدمة**: pyTelegramBotAPI (telebot)
- **معالجة الصور**: مكتبة Pillow لدمج الشعار مع الصور
