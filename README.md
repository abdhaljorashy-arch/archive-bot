# بوت المكتبة الإلكترونية للأرشفة والتوثيق
# Electronic Library Bot for Archiving and Documentation

بوت تليجرام مصمم لتسهيل عمليات الأرشفة والتوثيق، يتضمن نظام صلاحيات متعدد المستويات، تقارير إحصائية، ووحدة قولبة الصور.

## المميزات

- **قاعدة بيانات SQLite**: لتخزين بيانات المستخدمين والمحتوى المؤرشف
- **شجرة أزرار تفاعلية**: تنقل سهل عبر الأشهر الهجرية والأسابيع وأنواع الأنشطة
- **نظام صلاحيات (RBAC)**: تمييز بين المستخدمين العاديين والمشرفين
- **نظام الأرشفة**: رفع النصوص والوسائط (صور/فيديو/PDF) وحفظها
- **لوحة تحكم المشرف**: إشعارات فورية، سحب البيانات، إدارة المستخدمين
- **التقارير الإحصائية**: تقارير أسبوعية وشهرية
- **القولبة والدمج الذكي**: دمج النصوص مع الصور (محمي بكلمة مرور)

## التثبيت والإعداد

### 1. استنساخ المشروع

```bash
git clone https://github.com/abdhaljorashy-arch/archive-bot.git
cd archive-bot
```

### 2. إنشاء بيئة افتراضية

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 4. الإعداد

افتح ملف `config.py` وعدّل التالي:

- `TOKEN`: توكن البوت من BotFather
- `ADMIN_ID`: معرف تليجرام للمشرف الأول (رقم)
- `TEMPLATE_PASSWORD_SECRET`: كلمة مرور قوية لميزة القولبة
- `WATERMARK_PATH`: مسار ملف العلامة المائية (اختياري)

### 5. تشغيل البوت

```bash
python bot.py
```

سيتم إنشاء قاعدة البيانات (`archive_bot.db`) تلقائياً عند أول تشغيل.

## طريقة الاستخدام

- أرسل `/start` للبوت لبدء التفاعل
- اتبع الأزرار التفاعلية للتنقل بين القوائم
- **المستخدمون**: يمكنهم أرشفة المحتوى باختيار الشهر والأسبوع ونوع النشاط
- **المشرفون**: يمكنهم الوصول للوحة التحكم وإدارة المستخدمين وسحب البيانات

## ملاحظات تقنية

- **تخزين الملفات**: يحفظ البوت `file_telegram_id` بدلاً من تحميل الملفات لتوفير المساحة
- **إدارة المحادثات**: يستخدم `ConversationHandler` للتحكم في التنقل الشجري
- **قاعدة البيانات**: SQLite مع SQLAlchemy ORM
- **معالجة الصور**: مكتبة Pillow لدمج النصوص مع الصور
